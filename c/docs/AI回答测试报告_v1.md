## 错误分析

### 问题 20 — 边界处理不完善

**用户问题：** 「帮我写个 Python 脚本」
**AI 回答：** 「我不太确定您的问题与耳机有什么关系…」

**问题原因：** 系统提示词中边界规则的优先级不够高，AI 在不确定时选择了模糊回应而非明确引导。

**改进建议：** 在系统提示词中强化边界规则：「当用户问题明确不属于菠萝蓝牙耳机范畴时，必须礼貌但坚定地引导回服务范围。模板：『您好！我是菠萝耳机售后助手，关于 [用户问题主题] 的问题我可能帮不上忙。不过如果您有 AirPods 连接、续航、保修等方面的问题，我很乐意帮您解答！』」
