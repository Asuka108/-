
## 3. 聊天页交互状态

### 3.1 加载状态
- **页面加载：** 顶部显示骨架屏（灰色占位块），500ms 后显示欢迎消息
- **发送消息 loading：** 发送按钮变为灰色不可点击；消息列表底部出现 typing 动画气泡（三个灰点依次闪烁）

### 3.2 空状态
- **首次进入：** 显示 AI 欢迎消息 + 5 个快捷问题推荐
- **无历史对话：** 只显示欢迎消息

### 3.3 错误状态
- **网络断开：** 消息列表中显示红色提示条「网络连接失败，请检查网络后重试」，右侧有「重试」按钮
- **AI 超时：** 发送消息 15 秒后无响应，显示「AI 响应超时，请稍后重试」提示
- **服务器错误：** 显示「服务暂时不可用，请稍后再试」

### 3.4 键盘弹起
- 键盘弹出时，消息列表自动上推，输入区域保持在键盘上方
- 消息列表自动滚到底部

### 3.5 消息操作
- **长按消息：** 弹出菜单「复制」「反馈」（仅 AI 消息）
- **点击快捷问题：** 自动填入输入框并发送

## 6. 交互动效说明

1. **消息出现动画：** 新消息从下方淡入（opacity 0→1 + translateY 10rpx→0，300ms）
2. **发送按钮状态切换：** 输入框为空时半透明（opacity 0.4），有内容时完全显示（opacity 1）
3. **typing 动画：** 三个灰色圆点依次放大缩小（循环 1.2s）
4. **页面切换：** 右滑返回（iOS 风格手势）
5. **快捷问题点击反馈：** 点击时缩放至 95%（100ms），松手恢复
