---

## 1. 后台页面清单

| 页面 | 路由 | 功能说明 |
|------|------|---------|
| 知识库管理页 | pages/admin/admin | 查看/搜索/新增/编辑/删除知识条目 |
| 对话记录查看页 | pages/admin/conversations | 查看所有用户对话记录 |

**入口设计：** 小程序聊天页 → 长按标题栏 3 秒（隐藏入口，防止普通用户误入）

## 5. 后台界面设计规范

| 元素 | 规格 | 说明 |
|------|------|------|
| 页面背景 | #F2F2F7 | 与前台保持一致 |
| 卡片背景 | #FFFFFF | 白色卡片，圆角 12rpx |
| 表格行高 | 88rpx | 行内容垂直居中 |
| 操作按钮 | 编辑：#007AFF / 删除：#FF3B30 | 文字按钮，间距 16rpx |
| 弹窗遮罩 | rgba(0,0,0,0.5) | 半透明黑色 |
| 弹窗卡片 | #FFFFFF，圆角 16rpx | 居中显示，左右边距 48rpx |
| 表单标签 | #1C1C1E，28rpx | 左对齐 |
| 输入框 | #FFFFFF，边框 #C6C6C8，圆角 8rpx | 高度 72rpx |
| 文本域 | #FFFFFF，边框 #C6C6C8，圆角 8rpx | 高度 200rpx |
| 保存按钮 | #007AFF，白色文字，圆角 8rpx | 高度 72rpx |
| 取消按钮 | #8E8E93，文字按钮 | 高度 72rpx |

## 6. 管理员操作交互

| 操作 | 触发方式 | 确认机制 | 反馈 |
|------|---------|---------|------|
| 新增知识 | 点击「+ 新增」按钮 | 弹出空白表单 | 保存后 toast「新增成功」 |
| 编辑知识 | 点击行「编辑」按钮 | 弹出已填充表单 | 保存后 toast「修改成功」 |
| 删除知识 | 点击行「删除」按钮 | 弹出确认框「确定删除？」 | 确认后 toast「已删除」 |
| 搜索知识 | 输入框输入关键词 | 实时过滤列表 | 无结果时显示「未找到匹配项」 |
| 分类筛选 | 点击左侧分类名 | 过滤该分类条目 | 高亮选中分类 |
| 查看对话 | 点击对话条目 | 跳转详情页 | 加载对话消息列表 |
| 分页切换 | 点击上一页/下一页 | 加载对应页数据 | 当前页码高亮 |
